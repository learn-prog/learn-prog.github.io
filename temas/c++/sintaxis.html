C++

C++ es un lenguaje de programación orientado a objetos que toma la base del lenguaje C.

La intención de su creación fue el extender al exitoso lenguaje de programación C con mecanismos que permitieran la manipulación de objetos. En ese sentido, desde el punto de vista de los lenguajes orientados a objetos, el C++ es un lenguaje híbrido (¿por? fusionar con el siguiente).

Una variable, por lo general, como su nombre lo indica, puede variar su valor durante la ejecución del programa y también puede cambiar el tipo de dato que almacena estos lo veremos más adelante.

Te mostraremos la sintaxis básica del lenguaje de programación C++:

#include <iostream> --> Librería, hay varios tipos de librerías y se utilizan depende a lo que vayas a realizar, pero la más común y más utilizada es la librería iostream.

int n,f=1,i; --> Declarar en algoritmo y es la manera de declarar las variables donde el tipo de dato que se le agrega a la variable varia (int).

cout<<"Texto"<<endl; -->Es para mostrar un mensaje en pantalla, es decir escribir que sería en algoritmo.

cin>>Variable; -->Es para la entrada de datos que se almacenaran en la variable, que seria leer en algoritmo.

for(i=1;i<=n;i++){…} --> Repita para, el cual se abre o se comienza con llave y finaliza de la misma manera con llave.

while (i<=10) {…} --> Repita mientras, el cual se comienza con llave y finaliza de la misma manera con llave.

do{...} while(i<=10) --> Repita Hasta,Comienza con do y culmina con el while en donde se coloca la condición a verificar.

if (10>5) --> Si en algoritmo, el cual se comienza con llave y finaliza de la misma manera con llave.

else --> De lo contrario el cual se indiqca después de el if cuando la condición no se cumple de igual manera el bloque de instrucciones se comienza con llave { y se cierra de la igual manera }.

else if --> De lo contrario si.
